# 임상사진 에디터

치과 임상사진 기초 보정용 초경량 웹 에디터. **설치 없음 — 브라우저에서 바로.**

👉 **https://perioahn.github.io/photo-editor/**

## 지원 형식
JPEG · PNG · **NEF**(카메라 내장 풀사이즈 JPEG 기준 보정 — RAW 현상 아님).
출력은 항상 `_e.jpg`, NEF의 촬영일·카메라 정보도 보존.

## 기능
- 폴더 열기 → 필름스트립으로 수십 장 연속 보정 → **저장 후 다음** 한 번에
- 휠 줌 · 드래그 팬 · 90° 회전 · 좌우/상하 플립
- **📐 수평 맞추기**: 수평(수직)이어야 할 선을 드래그로 그으면 자동 각도 보정
- 미세 회전(±15°) · 밝기 · 대비 슬라이더 (라벨 더블클릭 = 리셋)
- 크롭 (자유 / 4:3 / 3:2, 3분할 격자)
- `\` 원본 비교 · Ctrl+Z 실행취소 · 이전 보정값 붙여넣기(Ctrl+V)

## 안전
- **원본은 절대 수정하지 않음** — 같은 폴더에 `이름_e.jpg`로 저장
- **EXIF 보존**: 촬영일·카메라 정보 유지 (사진 정리 프로그램과 호환)
- 모든 처리는 브라우저 안에서만 — 사진이 외부로 전송되지 않음

## 브라우저
Edge / Chrome / Whale (크로미움) 필요 — 폴더에 바로 저장하는 기능 때문.
다른 브라우저에서는 저장이 다운로드 폴더로 떨어짐.

## 단축키
| 키 | 동작 |
|---|---|
| ← / → | 이전 / 다음 사진 |
| R | 크롭 |
| Ctrl+] / Ctrl+[ | 90° 회전 |
| \\ | 원본 비교 토글 |
| Ctrl+S | 저장 |
| Ctrl+Shift+S | 저장 후 다음 |
| Ctrl+Z | 실행취소 |
| Ctrl+V | 이전 보정값 붙여넣기 |
| 0 | 화면 맞춤 |
